base-logger

Returns:
a Logger object with logging functions.
Type
module:base-logger~Logger
logger is a basic logger module used throughout the run scripts to log onto the node console or files. It takes a module name as reference and returns a logger object with log, info, warn and error methods.

LoggerSettings

The logger object used to configure loggers
Type:
  • object
Properties:
Name Type Argument Default Description
pattern regex <optional>
 \.*\ pattern is used to filter modules to log. Only moduleNames matching the pattern will be parsed through this setting
level string <optional>
 'log' the log level filters the actual module:base-logger~LoggingFunction in the module:base-logger~Logger object that are considered for logging. Those with priority lower than the passed level will be ignored.
appender string <optional>
 'console' appender defines the logging location.
